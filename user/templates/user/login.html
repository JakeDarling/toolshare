Beneficial to Everyone

Got tools laying around that you rarely use? Need to borrow a bandsaw for a project? ToolShare is here to help! ToolShare provides an easy-to-use interface for sharing tools within a community. Users in the same zip code can freely share items with other members.

How it Works

There are two options for sharing tools, you can either

  • Keep the tool at home or other private location and only share it with people who ask for it.
  • OR
  • Place the tool in a "community shed" where anyone who needs it can borrow it.

Anyone can add tools to a community shed. Creating a community shed makes you the manager of the shed. At community sheds, tools are free to come and go any time. Tools at these locations can be rented out just by clicking the borrow button.

Private sheds are typically places like your house or another location you privately own. Only the owner of a private shed can add tools to it and anyone wishing to borrow from a private shed has to first request permission from the shed owner.

Getting Started

The first thing to do is set up a community or private shed and add some tools to it. Then other users will be able to start borrowing your tools. If at any time you don't want others to borrow one of your tools, you can either remove that tool from the system or change its availability to unavailable.
